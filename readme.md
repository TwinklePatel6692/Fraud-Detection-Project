# 💳 Financial Fraud Detection System
## 📌 Overview
This project focuses on detecting fraudulent financial transactions using Machine Learning techniques. It aims to help financial institutions identify high-risk transactions in real-time and reduce fraud losses.
## 🎯 Problem Statement
Fraudulent transactions pose a major risk in the banking and financial sector. Traditional rule-based systems often fail to detect evolving fraud patterns.

This project uses a data-driven approach to classify transactions as **fraudulent or legitimate** based on historical data.
## 🚀 Solution
Built a Machine Learning classification model using Scikit-learn to predict fraudulent transactions. The system processes transaction data, applies feature engineering, and generates predictions with high accuracy.

A user-friendly interface is developed using Streamlit to allow real-time interaction.
## 🛠️ Tech Stack
**Language**: Python  
**Libraries**:Pandas, NumPy, Scikit-learn  
**Visualization:** Matplotlib / Seaborn  
**Deployment/UI:** Streamlit  
**Model Saving:** Joblib 

##  Features
- Real-time fraud prediction  
- Interactive Streamlit dashboard  
- Data preprocessing and feature engineering  
- Model training and evaluation  
- Easy-to-use interface for non-technical users
